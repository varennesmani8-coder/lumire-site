================================================================================

Windows Terminal or Command Prompt

================================================================================
STEP 2: Navigate to the project directory
================================================================================
cd "C:\Users\mangi\Documents\Agent Claude\Claude Code\Dropshipping\LUMIRÉ\lumire-site"

================================================================================
STEP 3: Verify Node.js and npm are installed
================================================================================
node --version
npm --version

(Should show v14+ and npm 6+)

================================================================================
STEP 4: Install Vercel CLI globally
================================================================================
npm install -g vercel

================================================================================
STEP 5: Deploy to Vercel production
================================================================================
vercel --prod

When prompted, answer:
- "Set up and deploy?" → Press 'y' and Enter
- "Which scope should we deploy to?" → Select your Vercel account (press up/down + Enter)
- "Link to existing project?" → Press 'n' and Enter (unless you already have one)
- "What is your project's name?" → Type 'lumire-site' and press Enter
- "In which directory is your code located?" → Just press Enter (default: .)

================================================================================
STEP 6: Deployment Complete!
================================================================================
Your site will be deployed to: https://lumire-site.vercel.app

You'll see output like:
  Production URL: https://lumire-site.vercel.app

2. Check that the page loads
3. Open browser developer tools (F12)
4. Check console for errors
5. Verify all images and styles load correctly
