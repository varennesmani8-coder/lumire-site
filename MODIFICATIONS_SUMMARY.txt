   ✓ Detects scroll direction by comparing current scroll position with previous position
   ✓ Hides header (translateY(-100%)) when scrolling down past 100px threshold

   ✓ Smooth CSS transition of 0.3s for hide/show effect
   ✓ Method is called in UIManager constructor

   - Header slides up when user scrolls down (past 100px)
